# my-reads-app

my-reads-app is a Reactjs project as books library. you can search in books and add and change your books status "want to read, currently reading or Read"

## Installation

```bash
npm install
```

if You are using yarn

```bash
yarn install
```

## Usage

In the project directory, run:

```bash
npm start
```

if You are using yarn

```bash
yarn start
```

## build

```bash
npm run build
```

if You are using yarn

```bash
yarn build
```
